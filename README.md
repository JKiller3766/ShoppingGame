## Delivery 03 - Shopping Game

![$(Game Title)](Screenshots/screenshot01.png "$(Game Title)")

### Description

You are and adventurer who found an old hut in the woods.
Buy whatever you need to go through your next adventure.

### Features

 - Buy and Sell items
 - Drag and Drop items between inventories to buy and sell.
 - Change between languages.

### Controls

Keyboard:
 - ENTER: Start and Restart the game
 - ESC: Quit game

Mouse:
 - LEFT CLICK: To interact with items and buttons.
 - Left Click over the health & hunger bar to lower them.
